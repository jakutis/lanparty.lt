# API package

An HTTP server written in Go language.
All logic, all error conditions, everything is extensively logged.

## Running

```bash
PORT=8080 OPENROUTER_API_KEY=... OPENROUTER_MODEL=... go run ./src
```

```bash
curl -XPOST localhost:8080/v1/representation -d '{"type":"html","spec":"a hello world web page"}'
```

## Verifying

See [./main/verifying.md](./main/verifying.md).

## Endpoints

All endpoints are prefixed with /v1.

### POST /representation

Request entity template:
```json
{
  "type": "html",
  "spec": "Some prompt"
}
```

Response entity is a file generated by an LLM via OpenRouter from the given spec and type.

The response sets a `Content-Disposition: attachment; filename="representation.<ext>"` header, where `<ext>` is the file extension for the requested `type` (e.g. `html` → `representation.html`).

## Generator

The representation endpoint delegates file generation to a [Generator](./main/generator.md): an LLM-backed component that produces raw file content for a given type and spec from the OpenRouter API.
